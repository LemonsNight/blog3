## 写在前面

最近想要搭建一个个人博客，在寻找合适的静态博客模板时，偶然发现了 [@tcdw](https://github.com/tcdw) 开源的 [Koi](https://github.com/tcdw/koi) 博客模板。使用了一段时间后，觉得非常满意，决定写篇文章表达一下感谢。

## 为什么选择 Koi

Koi 是一个基于 Astro 构建的静态博客模板，作者说是为他"不存在的赛博女儿"制作的。吸引我的地方有很多：

### 🎨 设计美观

Koi 的界面设计简洁大方，配色舒适，阅读体验很好。明亮和暗黑双主题自动切换，无论白天还是夜晚都能提供良好的浏览体验。

### 📱 响应式支持

在这个多设备并存的时代，响应式设计必不可少。Koi 完美支持各种尺寸的屏幕，无论是手机、平板还是桌面电脑，都能正常显示。

### ⚡ 性能优秀

Astro 的架构让 Koi 只需要加载很少的 JavaScript，页面加载速度非常快。这对于博客这种内容型网站来说非常重要。

### ♿ 可访问性好

Koi 的样式设计基本符合 WCAG 规范，对残障人士友好。这一点很多博客模板都忽略了，但 Koi 做得很好。

### 🔍 SEO 友好

内置了 sitemap 集成，搜索引擎优化做得很到位，有利于博客内容的传播。

## 技术栈

Koi 使用了现代化的技术栈：

- **Astro** - 通用内容性网站框架
- **Tailwind CSS** - 原子化 CSS 框架
- **Svelte** - 轻量级 UI 库
- **TypeScript** - 类型安全

这些技术都是当前前端开发的主流选择，社区活跃，文档完善。

## 使用体验

整个搭建过程非常顺利，按照 README 的说明操作即可：

```bash
pnpm install
pnpm dev
```

配置文件清晰易懂，很快就完成了个性化定制。MDX 的支持让写博客变得更加灵活，可以在文章中嵌入交互式组件。

## 致谢

非常感谢 @tcdw 开源这么优秀的项目。开源精神让世界变得更美好，也希望 Koi 能够帮助到更多想要搭建个人博客的人。

如果你也在寻找一个简洁、快速、美观的博客模板，不妨试试 [Koi](https://github.com/tcdw/koi)。
